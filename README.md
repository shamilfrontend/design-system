# Vue3 Design System

Публичная библиотека компонентов на Vue 3 с документацией и Storybook.

## Стек

- Vue 3 + TypeScript strict
- Vite library mode
- Storybook (Vue3 + Vite)

## Команды

```bash
yarn
yarn typecheck
yarn lint
yarn build
yarn storybook
```

## Структура

- `src/components` — компоненты системы
- `src/tokens` — токены и тема
- `.storybook` — конфигурация Storybook
- `docs` — документация использования
