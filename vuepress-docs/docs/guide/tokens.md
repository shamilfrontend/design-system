# Токены

Базовые design tokens находятся в [`src/vars.scss`](https://github.com/shamilfrontend/design-system/blob/main/src/vars.scss).

## Радиусы

- `--border-radius-base`: 6px
- `--border-radius-lg`: 8px

## Типографика

- `--font-size-base`: 14px
- `--line-height-base`: 22px
- `--font-weight-base`: 400
- `--font-weight-medium`: 500
- `--font-weight-bold`: 600

Шрифт: **Urbanist** (подключается через `@shamilfrontend/design-system/fonts`).

## Layout

- `--layout-gutter`: 32px

## Анимации

- `--transition-spline-base`: cubic-bezier(0.645, 0.045, 0.355, 1)

## Подключение

```js
import '@shamilfrontend/design-system/styles';
```
