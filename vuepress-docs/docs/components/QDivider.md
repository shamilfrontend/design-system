# QDivider

Разделитель контента.

## Props

### direction

- Type: `String`
- Default: `'horizontal'`
- horizontal | vertical

### contentPosition

- Type: `String`
- Default: `'center'`
- left | center | right

## Пример

```vue
<template>
  <q-divider>OR</q-divider>
</template>

<script setup lang="ts">
import { QDivider } from '@shamilfrontend/design-system';
</script>
```
