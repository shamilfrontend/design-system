# QList

Список элементов.

## Props

### bordered

- Type: `Boolean`
- Default: `false`
- Рамка

### size

- Type: `String`
- Default: `'medium'`
- small | medium | large

## Пример

```vue
<template>
  <q-list bordered><q-list-item>Item</q-list-item></q-list>
</template>

<script setup lang="ts">
import { QList } from '@shamilfrontend/design-system';
</script>
```
