# QText

Typography: inline текст.

## Props

### type

- Type: `String`
- Default: `'default'`
- default | secondary | success | warning | danger

## Пример

```vue
<template>
  <q-text type="secondary">Muted text</q-text>
</template>

<script setup lang="ts">
import { QText } from '@shamilfrontend/design-system';
</script>
```
