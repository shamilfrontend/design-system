# QBackTop

Кнопка прокрутки наверх.

## Props

### visibilityHeight

- Type: `Number`
- Default: `200`
- Порог scrollY

### right

- Type: `Number`
- Default: `40`
- Отступ справа

### bottom

- Type: `Number`
- Default: `40`
- Отступ снизу

## Пример

```vue
<template>
  <q-back-top />
</template>

<script setup lang="ts">
import { QBackTop } from '@shamilfrontend/design-system';
</script>
```
