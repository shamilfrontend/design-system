# QAvatar

Аватар пользователя.

## Props

### size

- Type: `String`
- Default: `'medium'`
- small | medium | large

### src

- Type: `String`
- Default: `null`
- URL изображения

## Пример

```vue
<template>
  <q-avatar>AB</q-avatar>
</template>

<script setup lang="ts">
import { QAvatar } from '@shamilfrontend/design-system';
</script>
```
