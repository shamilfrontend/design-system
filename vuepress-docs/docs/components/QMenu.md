# QMenu

Меню для QDropdown.

## Props

### mode

- Type: `String`
- Default: `'vertical'`
- vertical | horizontal

## Пример

```vue
<template>
  <q-menu><q-menu-item>Item</q-menu-item></q-menu>
</template>

<script setup lang="ts">
import { QMenu } from '@shamilfrontend/design-system';
</script>
```
