# QTitle

Typography: заголовок.

## Props

### level

- Type: `Number`
- Default: `2`
- 1–6 → h1–h6

## Пример

```vue
<template>
  <q-title :level="2">Heading</q-title>
</template>

<script setup lang="ts">
import { QTitle } from '@shamilfrontend/design-system';
</script>
```
