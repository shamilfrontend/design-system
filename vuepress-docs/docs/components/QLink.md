# QLink

Ссылка с темой DS.

## Props

### href

- Type: `String`
- Default: `''`
- URL

### target

- Type: `String`
- Default: `null`
- target атрибут

### disabled

- Type: `Boolean`
- Default: `false`
- Отключена

## Пример

```vue
<template>
  <q-link href="https://example.com">Link</q-link>
</template>

<script setup lang="ts">
import { QLink } from '@shamilfrontend/design-system';
</script>
```
