# QAffix

Sticky wrapper.

## Props

### offsetTop

- Type: `Number`
- Default: `0`
- top offset px

## Пример

```vue
<template>
  <q-affix :offset-top="64"><q-button>Sticky</q-button></q-affix>
</template>

<script setup lang="ts">
import { QAffix } from '@shamilfrontend/design-system';
</script>
```
